A browser-based tool to inspect text from PDF documents. It uses [PDF.js](https://mozilla.github.io/pdf.js/) to render PDF pages, highlights all visible text, and displays detailed font and position metadata for selected text.

## 🔍 Features

- Load PDF from a public **URL**
- Automatically highlight **all visible text**
- Click on any text highlight to:
  - View text content
  - Font size in pixels
  - Font family (from PDF metadata)
